# Catálogo de Cursos - Sistemas (Caso Práctico)

Esta es una mini-aplicación web desarrollada como parte del "Caso Práctico – Revisión grupal de cronogramas y seguimiento".

El objetivo de este artefacto es servir como un entregable simple para practicar la coordinación, el seguimiento de tareas y la validación del "Definition of Done" (DoD).

## Cómo Ejecutar el Proyecto

Este es un proyecto web estático (JS, HTML, CSS). No requiere un proceso de compilación ni dependencias complejas.

La forma más sencilla de ejecutarlo es:

1.  Asegúrate de tener los tres archivos en la misma carpeta:
    * `index.html`
    * `styles.css`
    * `app.js`
2.  Haz doble clic en el archivo **`index.html`**.

Esto abrirá el proyecto directamente en tu navegador web predeterminado y todo funcionará.

### Método Alternativo (Servidor Local)

Si prefieres (o si en el futuro añades funciones que lo requieran), puedes usar un servidor local simple. Si tienes Node.js instalado, puedes usar `serve`:

```bash
# 1. Instala 'serve' globalmente (solo la primera vez)
npm install -g serve

# 2. Desde la carpeta donde están tus archivos, ejecuta:
serve

# 3. Abre la dirección que te indica (usualmente http://localhost:3000)
